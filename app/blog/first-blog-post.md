# Building My Personal Site with Next.js and Vercel

I've been wanting a place to call my own on the internet for a while now. Somewhere to share what I'm working on, thinking about, and the occasional hot take on data analytics and technology.

This site is built with Next.js and deployed on Vercel. I'm using Vercel Blob for storing blog posts as simple markdown files, which makes it incredibly easy to publish new content - just upload a file and it's live.

The design is intentionally minimal with a dark theme and some subtle gradient effects. I wanted something that felt professional but also had a bit of personality. The animated "PM" logo with the cycling x's is my little touch - a nod to my x.com handle.

Tech stack highlights:
- Next.js 16 with App Router
- Tailwind CSS for styling
- Vercel Blob for content storage
- TypeScript throughout

More to come. Topics will range from data engineering deep dives to thoughts on AI in the workplace. Thanks for stopping by.
